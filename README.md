# 🏋️ Gym Web Project

## 📌 Descripción
Este proyecto es una aplicación web para un gimnasio que incluye un sistema de registro e inicio de sesión de usuarios. Permite a los usuarios crear una cuenta, iniciar sesión y acceder a una sección principal protegida.

El objetivo del proyecto es practicar desarrollo web full-stack utilizando HTML, CSS, PHP y MySQL.

---

## ⚙️ Tecnologías utilizadas
- HTML5
- CSS3
- PHP
- MySQL
- Git / GitHub

---

## 🔐 Funcionalidades
- Registro de usuarios
- Inicio de sesión (login)
- Validación de datos de usuario
- Contraseñas encriptadas con `password_hash()`
- Verificación de contraseña con `password_verify()`
- Conexión a base de datos MySQL
- Sesiones con PHP (`$_SESSION`)

---

## 🗄️ Base de datos
La base de datos contiene una tabla llamada `users` con los siguientes campos:

- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- username (VARCHAR)
- email (VARCHAR)
- password (VARCHAR)
- created_at (TIMESTAMP)
